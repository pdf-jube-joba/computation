[[Model]]

変数の集合 \(V\) を固定しておく。
文法と意味論を別々に定義するほどでもないかな。

環境
  : \(V \to \Nat\) のこと

代入
  : 環境 \(s\) と変数 \(x\) および自然数 \(n\) に対して、
    環境 \(s[x \rightarrow n]\) を、変数 \(x\) に対しては \(n\) とし、
    それ以外の変数 \(y\) では \(s(y)\) として定める。

goto プログラムは次のテーブルで定義される文の列のこと。

| 構文 | 意味 |
| ---- | ---- |
| \(\T{inc} (s \in V)\) | \(s\) に \(1\) を加算する |
| \(\T{dec} (s \in V)\) | \(s\) から \(1\) を減算する |
| \(\T{clr} (s \in V)\) | \(s\) を \(0\) にする |
| \(\T{cpy} (s_1 \in V) (s_2 \in V)\) | \(s_2\) の値を \(s_1\) に代入する。 |
| \(\T{ifnz} (s \in V) (n \in \N)\) |\(s\) に入っている値が \(0\) のとき、 \(n\) 行目のコードに移る。|
